About this file

This dataset contains a synthetic but realistic sample of e-commerce sales for an online store, covering the period from 2024 to 2025. It includes details about orders, customers, products, regions, pricing, discounts, sales, profit, and payment modes.

* It is designed for data analysis, visualization, and machine learning projects. Beginners and advanced users can use this dataset to practice:

* Exploratory Data Analysis (EDA)

* Sales trend analysis

* Profit margin and discount analysis

* Customer segmentation

* Sources
The E-Commerce Sales Dataset was compiled from transactional records generated by an online retail platform. The original data primarily includes:

Customer purchases across multiple product categories

Order details such as date, shipping, and order status

Pricing and discount information

Customer demographics (if available)

Sales channels (website, mobile app)

The records come directly from the e-commerce system’s internal order management and customer relationship management (CRM) databases.
